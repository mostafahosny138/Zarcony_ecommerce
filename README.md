
## Zarcony Ecommerce Task

1- clone the project

2-cd Zarcony_ecommerce

3- composer update

4- php artisan migrate --seed

5- php artisan serve

6-  admin can access through api with attach postman collection (Zarcony.postman_collection) existed on the root of project

7-run the feature/unit -> php artisan test and you can set configuration of test env on phpunit.xml

8- you can access erd on root of project exist screenshot with name Zaecony Ecommerce ERD 


php version 8.1.5

laravel framework 9.19
